# Qt记事本

这是一个使用Qt和C++开发的简单记事本应用程序。

## 功能特性

- 创建、打开、编辑和保存文本文件
- 支持基本的文本编辑功能(剪切、复制、粘贴等)
- 字体设置
- 打印功能

## 环境要求

- Qt 6.8.0 或更高版本
- C++17 兼容的编译器
- CMake 3.16 或更高版本

## 编译和运行

1. 克隆此仓库:
   ```
   git clone https://github.com/your-username/qt-notepad.git
   cd qt-notepad
   ```

2. 创建构建目录:
   ```
   mkdir build
   cd build
   ```

3. 运行CMake:
   ```
   cmake ..
   ```

4. 编译项目:
   ```
   cmake --build .
   ```

5. 运行应用程序:
   ```
   ./notepad
   ```

## 使用说明

- 使用菜单栏或工具栏上的按钮来执行各种操作
- 文件菜单: 新建、打开、保存、打印文件
- 编辑菜单: 执行剪切、复制、粘贴等操作
- 格式菜单: 更改字体设置
